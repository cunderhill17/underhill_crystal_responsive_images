# Responsive Images Assignment
This repository is for responsive images assignment in the Web Development 1 Course in IDP3 @ Fanshawe in which we had to make 3 different sets of images responsive using the different methods taught in class. 

- Using the `img` tag and srcset
- Using the `picture` tag
- and changing out the background images using `media queries` 

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open the `index.html` file in the browser of your choice.
